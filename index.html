¿Qué es HTML5?

HTML5 es la versión más reciente del lenguaje HTML, que se utiliza para crear y estructurar páginas web. Sirve para organizar el contenido de un sitio, como textos, imágenes, enlaces y videos. Es importante porque permite crear páginas modernas, más rápidas y compatibles con diferentes dispositivos como celulares, tablets y computadoras sin necesidad de plugins adicionales.

Estructura básica

<!DOCTYPE html> <html> <head> <title>Mi pagina</title> </head> <body> </body> </html>

La estructura básica de HTML5 es la base de cualquier página web. La etiqueta DOCTYPE indica que se está usando HTML5. La etiqueta html contiene todo el contenido del sitio. En head se colocan configuraciones como el título o enlaces a archivos CSS. En body va todo lo visible para el usuario, como textos, imágenes y secciones.

Etiquetas principales

h1 - h6

Se utilizan para títulos y subtítulos, organizando la información por niveles de importancia.

p

Se usa para escribir párrafos de texto dentro de la página.

img

Permite insertar imágenes mediante la ruta del archivo.

a

Sirve para crear enlaces que llevan a otras páginas o sitios web.

button

Se utiliza para crear botones que pueden ejecutar acciones.

ul / li

Se usan para crear listas de elementos, ordenadas o desordenadas.

HTML5 semántico

header

Representa el encabezado de la página o de una sección.

nav

Se utiliza para agrupar enlaces de navegación.

main

Contiene el contenido principal del sitio web.

section

Define una sección dentro del documento.

article

Representa contenido independiente como artículos o publicaciones.

footer

Es el pie de página donde se coloca información adicional o créditos.

Formularios

Los formularios permiten al usuario ingresar información en una página web. Se utilizan para registros, inicios de sesión o envío de datos. Algunas etiquetas importantes son form, label, input, textarea y button, las cuales ayudan a capturar y enviar información.

Multimedia

HTML5 permite agregar contenido multimedia como imágenes, audio y video directamente en la página web. Esto facilita la creación de sitios más interactivos sin necesidad de programas externos.

Imágenes

Se agregan con la etiqueta img y permiten mostrar contenido visual.

Audio

La etiqueta audio permite reproducir sonidos dentro de la página.

Video

La etiqueta video permite mostrar videos sin usar reproductores externos.

Accesibilidad

El atributo alt en imágenes ayuda a describirlas para personas con discapacidad visual.
