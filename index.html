¿Qué es HTML5?

HTML5 es la versión más reciente del lenguaje HTML, que se utiliza para crear y estructurar páginas web. Sirve para organizar contenido como textos, imágenes, enlaces y videos. Es importante porque permite crear sitios modernos, rápidos y compatibles con diferentes dispositivos.

Estructura básica

<!DOCTYPE html> <html> <head> <title>Mi pagina</title> </head> <body> </body> </html>

La estructura básica de HTML5 es la base de cualquier página web. DOCTYPE indica el tipo de documento, html contiene todo el contenido, head incluye configuraciones y body muestra lo visible al usuario.

Etiquetas principales

h1 - h6

Se utilizan para títulos y subtítulos.

p

Permite escribir párrafos de texto.

img

Sirve para agregar imágenes.

a

Se usa para enlaces.

button

Permite crear botones interactivos.

ul / li

Se usan para listas.

HTML5 semántico

header

Encabezado del sitio.

nav

Menú de navegación.

main

Contenido principal.

section

Secciones del contenido.

article

Contenido independiente.

footer

Pie de página.

Formularios

Los formularios permiten ingresar datos del usuario como nombre, correo o comentarios. Se usan en registros, contacto y encuestas.

Multimedia

HTML5 permite agregar imágenes, audio y video directamente en la página, mejorando la experiencia del usuario.

Imágenes

Se agregan con img.

Audio

Se usa la etiqueta audio.

Video

Se usa la etiqueta video.

Accesibilidad

El atributo alt describe imágenes.
